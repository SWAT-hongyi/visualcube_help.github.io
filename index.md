# visual cube中文帮助文档
visual cube是一个可以生成魔方状态的网站，拥有繁多的功能。
网址为：<http://cube.rider.biz/visualcube.png?>
官方帮助文档（英文）<http://cube.rider.biz/visualcube.php>
## 参数使用
在网址http://cube.rider.biz/visualcube.png?后添加。参数间用 ***&*** 分割。
